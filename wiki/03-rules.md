# 03. AI 이용 전 5가지 수칙

AI 사용 시 아래 5가지 수칙은 꼭 지켜주세요.

### RULE 01 — 회사 자료는 회사 공식 AI에서 다룹니다
회사가 도입한 공식 AI(M365 Copilot, Claude Team)에서 회사 자료를 다룹니다. 상용 AI에는 회사 자료를 올리지 않습니다.

### RULE 02 — 인증정보는 어떤 AI에도 입력하지 않습니다
비밀번호·API Key·Token은 어떤 AI에도 입력하지 않습니다. 인증정보는 Vault 등 별도 보관소를 사용합니다.

민감정보 관련 문의는 **Slack #help_정보보안**으로 부탁드립니다.
{: .note }

### RULE 03 — 민감한 정보는 먼저 가립니다
개인정보·고객정보·계약정보는 그대로 입력하지 않습니다. 필요한 경우 이름·번호 등을 다른 표현으로 바꿔 사용합니다.

**예시:**
- 홍길동 → 고객 A
- 전화번호 → 010-****-****

### RULE 04 — 상용 AI는 학습 옵션을 끄고 사용합니다
상용 AI는 기본값이 학습 ON인 경우가 많습니다. 설정 가이드에 따라 학습 옵션을 OFF로 변경한 뒤 사용합니다.

### RULE 05 — AI 결과물은 사람이 검토한 뒤 사용합니다
AI는 그럴듯하지만 틀린 답(환각)을 만들 수 있습니다. 코드·문서·분석 결과를 외부에 공유하기 전에는 사람이 한 번 검토합니다.

고객 안내·법률 문서·보안 관련 작업은 반드시 담당자가 최종 확인합니다. **AI 결과물의 최종 책임은 사용자에게 있습니다.**
{: .warning }
