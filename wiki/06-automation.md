# 06. AI가 직접 작업할 때는 한 번 더 확인합니다

AI는 실제 작업도 수행할 수 있습니다. 실행 전 한 번 더 확인합니다.

## AI 자동화란?
AI가 사용자를 대신해 메일 발송·일정 등록·파일 정리 같은 작업을 수행하거나, 여러 서비스를 연결해 작업을 자동으로 처리하는 기능입니다. 편리하지만 잘못된 작업도 실행될 수 있습니다.

### 01. 사람이 검증합니다
AI 자동화는 바로 운영에 적용하지 않고, **테스트를 통해 정상 동작 여부를 먼저 확인**합니다. 적용 후에도 정상 동작 여부를 지속적으로 확인합니다.

### 02. 작업은 더 신중하게
**발송·결제·삭제·서명**은 한 번 실행되면 되돌리기 어렵습니다. 이런 작업은 사람이 최종 승인 후 실행합니다.

### 03. 권한은 필요한 만큼만 줍니다
AI에게 메일·캘린더·파일 접근 권한을 줄 때 **읽기·쓰기·삭제 중 꼭 필요한 것만** 허용합니다. 사용 후에는 권한을 제거합니다.

## 사용 예시

| 구분 | 사례 |
|---|---|
| 권장 | **회의록 자동 작성**: AI가 회의 녹취를 요약해 회의록 초안을 만들고, 사람이 검토한 뒤 참석자에게 발송합니다. |
| 주의 | **고객 응대 자동 회신**: AI가 고객 문의에 자동 회신하는 경우, 정확성·톤·법적 표현을 사람이 마지막에 확인합니다. |
| 권장 | **일정 후보 제안**: AI가 캘린더를 확인해 가능한 시간대 후보를 제안하고, 사람이 최종 선택해 등록합니다. |
| 주의 | **파일 정리·삭제 위임**: AI에게 "오래된 파일 정리"시 1차적으로 정리 목록을 받고 삭제를 승인합니다. |
